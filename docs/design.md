# ConfigDialog 设计规范文档

> 用于将此配置对话框的视觉风格迁移至其他项目。
> 基于 PicScreenSaver Runtime 项目的 `ConfigDialog.xaml` 提取。

---

## 1. 整体布局

| 区域 | 高度 | 说明 |
|------|------|------|
| 标题栏 | 44px | 白色背景，底部 1px 分割线 |
| 内容区 | `*` (剩余) | 左右双栏布局，间距 16px |
| 底部栏 | 44px | 白色背景，顶部 1px 分割线 |

- **窗口样式**：无边框（`WindowStyle="None"`）+ `AllowsTransparency="True"`
- **启动位置**：屏幕居中
- **置顶**：`Topmost="True"`
- **拖拽**：标题栏支持鼠标拖拽移动窗口

### 内容区网格

```
┌──────────────────────────────────────────┐
│              标题栏 (44px)                  │
├──────────────────┬──┬────────────────────┤
│                  │16│                     │
│    左栏 (1:1)    │px│     右栏 (1:1)     │
│                  │  │                     │
├──────────────────┴──┴────────────────────┤
│              底部栏 (44px)                  │
└──────────────────────────────────────────┘
```

左右栏间距 16px，左右各 Margin 24px。

## 3. 字体

| 属性 | 值 |
|------|-----|
| 西文/等宽字体 | **Consolas**（数值标签、计数） |
| 中文字体 | **Noto Sans SC**（全部中文文本） |
| 字号范围 | 10.5px – 22px |

字体回退：WPF 中 `FontFamily="Noto Sans SC"` 会自动回退到系统默认中文字体。

---

## 4. 圆角

| 元素 | 圆角 | 说明 |
|------|------|------|
| 内容卡片 | `10px` | 左右双栏卡片外框 |
| 卡片底部 | `0,0,10,10` | 仅左下/右下圆角 |
| 数值标签 | `4px` | 时长数值标签背景 |
| 预览区域 | `6px` | 过渡效果预览容器 |

**注意：** 卡片底部使用 `CornerRadius="0,0,10,10"` 配合 `ClipToBounds="True"` 保证内容不溢出圆角区域。卡片顶部的 Section Header 不做上圆角（直边 + 分割线），模拟层叠效果。

## 5. 分割线

全部使用 1px 实线 `#CDD0DA`（`BorderThickness` 指定方向）：

| 位置 | 方向 | 示例 |
|------|------|------|
| 标题栏底部 | 下 | `0,0,0,1` |
| 各 Section 底部 | 下 | `0,0,0,1` |
| 预览区上方 | 上 | `0,1,0,0` |
| 底部栏顶部 | 上 | `0,1,0,0` |
| 卡片外框 | 四边 | `1`（`BorderThickness="1"`） |

## 7. 组件级样式

### 7.1 标题栏

- 白色背景 + 底部 1px 分割线
- 标题文字居中：`PicScreenSaver`（12.5px, Medium, `#555A70`）
- 右上角关闭按钮：36×44px 点击区域，透明背景
  - 关闭图标：`×` 形两条线，`Stroke="#555A70"`，`StrokeThickness="1.5"`，圆头端点
  - 图标尺寸 10×10px，通过 `Viewbox` 保持比例
- 双击标题栏 → 关闭窗口

### 7.2 Slider（滑块）

```
Foreground="#5570E8"   ← 主题色滑块
Margin="0,6,0,0"       ← 距上方描述 6px
```

各 Slider 参数：

| Slider | 范围 | 步进(SmallChange) | 大跳(LargeChange) | Tick频率 | 吸附 |
|--------|------|-------------------|-------------------|----------|------|
| 每张展示时长 | 1–60s | 0.5 | 1 | 0.5 | 是 |
| 过渡动画时长 | 0.3–5s | 0.1 | 0.5 | 0.1 | 是 |

### 7.3 数值标签

```
背景：#E8E9EF
圆角：4px
内边距：6px 水平 + 2px 垂直
文字：Consolas 12px SemiBold #5570E8
格式：F1（一位小数）
右边跟随单位文字：10.5px #555A70
```

### 7.4 RadioButton（播放顺序）

- 前景色 `#555A70`
- 字号 12px
- `Cursor="Hand"`
- 间距：右侧距下一个 12px

### 7.5 过渡效果预览

```
容器大小：220×160px
背景：#E8E9EF
圆角：6px
居中显示（VerticalAlignment="Center" HorizontalAlignment="Center"）
两张图片叠加堆叠，通过 Storyboard 做过渡动画预览
```

预览循环播放：当前效果预览完成后自动重新开始。

### 7.6 效果选择区（右栏）

- 标题栏右侧包含"全选"和"清空"按钮
- 效果以 Checkbox 列表展示，2 列 `UniformGrid`
  - 文字 `#1A1D2A`，字号 11px
  - 高度 22px，间距 2×1px
  - `Cursor="Hand"`
- 选中/取消时实时更新底部信息区
- 全部取消时显示 `—` / "未选择任何效果"

### 7.7 底部信息栏（左栏底部）

| 区域 | 内容 |
|------|------|
| 左侧 | 当前选中效果名（13px SemiBold）+ 描述（11px, 自动换行） |
| 右侧 | 已选数量（22px SemiBold Consolas 主题色）+ "已选" 标签（10.5px 辅助色） |

背景 `#F5F6FA`，仅左下/右下圆角 `0,0,10,10`。

### 7.8 主操作按钮

| 属性 | 值 |
|------|-----|
| 宽度 | 72px（确定）/ 110px（安装） |
| 高度 | 28px |
| 圆角 | 0（无圆角，系统默认） |
| 文字 | 11px SemiBold |
| 内边距 | 无额外 |

### 7.9 图标

- 窗口图标：从嵌入资源加载（`Resources/sys.png`)
- 不在 XAML 中设置，在 code-behind 的 `ConfigDialog` 构造函数中通过 `Icon = img` 赋值

## 9. 代码逻辑行为（用于复现交互）

| 行为 | 说明 |
|------|------|
| 窗口关闭 | 触发 `Environment.Exit(0)`（通过 `Closing` 事件） |
| 拖拽 | 标题栏 `MouseLeftButtonDown` → `DragMove()` |
| 双击关闭 | 标题栏双击 → `Close()` |
| 安装按钮 | 提权 `cmd.exe /c copy` 复制到 System32，设置注册表 `SCRNSAVE.EXE` |
| 确定按钮 | 保存配置到 JSON → 关闭窗口 |
| 配置回读 | 构造函数中从 `ScreensaverConfig` 对象设置各控件初始值 |

## 10. 迁移清单

迁移至新项目时需提供的资源：

| 项 | 说明 |
|----|------|
| Noto Sans SC 字体 | 确保运行时可用，或替换为等效中文字体 |
| Consolas 字体 | 等宽数字显示，可替换为 JetBrains Mono / SF Mono 等 |
| 图标资源 | 窗口图标（png 嵌入资源） |
| 预览图片 | `LoadImageOrDefault(0/1, ...)` —— 两张示例图 |
| ScreensaverConfig | 配置数据模型（包含 DisplayDuration, TransitionDuration, ShuffleImages, SelectedEffects 等字段） |
